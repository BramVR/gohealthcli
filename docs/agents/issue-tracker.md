# Issue Tracker: GitHub

Issues and PRDs for this repo live in GitHub Issues for `BramVR/gohealthcli`.
Use the `gh` CLI from this checkout.

## Conventions

- Create an issue: `gh issue create --repo BramVR/gohealthcli --title "..." --body-file PATH`.
- Read an issue: `gh issue view NUMBER --repo BramVR/gohealthcli --comments`.
- List issues: `gh issue list --repo BramVR/gohealthcli --state open`.
- Comment on an issue: `gh issue comment NUMBER --repo BramVR/gohealthcli --body-file PATH`.
- Apply or remove labels with `gh issue edit NUMBER --repo BramVR/gohealthcli --add-label "..."` or `--remove-label "..."`.
- Close with `gh issue close NUMBER --repo BramVR/gohealthcli --comment "..."`.

When a skill says "publish to the issue tracker", create a GitHub issue.
